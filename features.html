🚀 Professional Modes & Utility Tools

🧊 Isometric Drawing: Smart grid system locked to 30° and 150° for creating 3D perspectives in a 2D environment.
🧩 Boolean Operations: Perform mathematical operations on shapes: Union, Difference, or Intersection.
📊 Tables & Settings: Add technical tables with customizable rows, columns, and cell content to your workspace.
💥 Explode & Join: Break complex polylines into independent lines or combine open lines into a single continuous object.
🌙 Dark Theme Support: Professional interface option to reduce eye strain and provide comfort in low-light conditions.
📐 Ortho Mode: Ensure precision by forcing drawings to only horizontal (0°, 180°) and vertical (90°, 270°) axes.
⌨️ Keyboard Support: Comprehensive keyboard shortcuts providing instant one-key access to drawing and editing commands.
🔍 Filtering: Gain speed in complex projects by selecting only specific object types (e.g., Select Only Text, Select Only Circles).
