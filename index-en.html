Comprehensive and Fast 2D CAD Technical Drawing Capabilities

You can quickly detail your 2D technical drawings or illustrations with TamgamCAD's advanced tools. You can set the color, thickness, and line type of your drawings from the properties section, and later change their styles or transfer them to another object. Thanks to hatching options, you can draw detailed technical illustrations, and with precise measurement tools, you can find out the real-life equivalent of your drawings. You can quickly add the drawings you customized and detailed to the library using block creation for later use. To use the created block in different layers, you can use the project library; to use it in other projects, you can use the global library. For more details about block settings and other drawing tools, you can check out the user manual. While providing users with the basic features required for technical drawing/illustration, TamgamCAD also prioritizes speed and usability. One of the most basic CAD tools, layer management, allows users to organize and manage objects in the drawing according to user-defined properties. This feature, which is important for complex engineering and architectural drawing, increases efficiency and allows the project to grow systematically.

Frequently Asked Questions (FAQ)

Is an internet connection required to use TamgamCAD?

No, an internet connection is not mandatory. Thanks to its local workspace feature, TamgamCAD stores all your projects and drawings on your device, allowing you to work completely offline.

Can I use the Pro features I purchased on Google Play on the Microsoft Store?

No, the software's features vary depending on the platform it is published on. If the features included in your purchased subscription are not offered on the other platform, you cannot use those features. However, you can log in with the same email address.

Is it easy to use on tablets and touchscreens?

Absolutely! Our interface is specifically optimized for touchscreens. You can zoom/pan with two fingers and create drawings with millimeter precision using your stylus.

Is there an educational or trial version available?

Yes, we offer a 7 or 14-day free trial version (depending on the selected subscription) where all features are unlocked so you can test the program. You can download it via Google Play and try it immediately. You can easily use the software with in-app guidance without the need for complex training processes.
