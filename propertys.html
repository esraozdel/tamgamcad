Professional 2D CAD Features

TamgamCAD provides all the advanced technical drawing, modification, and Boolean-based drafting tools you need for your engineering and architectural projects on a single platform.

📁 File Management & Export

  • ✔️ Save projects safely and locally in .json format
  • ✔️ Insert and trace over JPG/PNG reference images
  • ✔️ Export to SVG and PDF for high-resolution vector presentations and prints
  • ✔️ Industry-standard DXF export support for seamless integration with CNC, laser cutters, and other CAD software
  • ✔️ Lossless PNG export for quick project sharing

✏️ Advanced Drawing Tools

  • ✔️ Continuous Polyline and Freehand drawing tools
  • ✔️ Basic Geometry: Circle, Arc, Ellipse, Polygon, and Donut
  • ✔️ Precise Geometry: Tan-Tan-Radius (TTR) and Tangent Lines
  • ✔️ Ortho Mode for axis-locked technical drafting
  • ✔️ Automatic Object Snapping (Osnap: Endpoint, Midpoint, Center, Intersect)

🧩 Modification & Boolean Operations

  • ✔️ Trim, Extend, Scale, and Resize elements
  • ✔️ Offset, Mirror, and Rotate
  • ✔️ Polar and Rectangular Array distributions
  • ✔️ Chamfer, Fillet, Group, and Explode
  • ✔️ Boolean Tools: Union, Difference, Intersection, XOR, and Boundary Wrap

📐 Dimensioning & Annotation

  • ✔️ Linear, Angular, Radius, Diameter dimensions and Area Calculation
  • ✔️ Hatching with diverse patterns (Solid, Lines, Honeycomb, Brick, Concrete, etc.)
  • ✔️ Descriptive plain text and Leader lines with arrows
  • ✔️ Customizable Property and Detail Tables for project analysis
  • ✔️ Precise Scale adjustments for real-world representation

🎯 Layers, Library & Viewport

  • ✔️ Dynamic Layer Management and Match Properties tool
  • ✔️ Create reusable Blocks with the Global Library
  • ✔️ Isometric Grid providing a 3D illusion in a 2D space
  • ✔️ Dedicated Dimension Input Panel tailored for touchscreens and tablets
  • ✔️ Eye-friendly Dark/Light Theme and Multi-Language Support (TR/EN)

An optimized, fast, and secure 2D CAD solution for professionals.
